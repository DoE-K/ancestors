# Ancestors

A playable Unity game that I am developing completely solo.

Genre: Top-Down Crafting RPG 
Engine: Unity  
Platform: Windows, MacOS 

Playable on itch.io: https://myst1ceast.itch.io/ancestors

---

## Features

- Player movement with physics
- Craft system 
- Animals with different behaviors (work in progress)
- Highscore system
- UI (work in progress)
- simple menu music
- Custom pixel art (work in progress)

---

## Project Structure

All gameplay-related scripts are located in:

Assets/
└── Scripts/
